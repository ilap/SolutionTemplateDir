# Skeleton README.md for Xamarin projects #

Skeleton directory structure of Xamarin projects

``` bash
.
├── arts
│   ├── icons
│   └── screenshots
├── docs
│   └── components
├── nuspec
├── README.md
├── readme.txt
├── references
├── samples
├── scripts
├── src
│   └── SharedAssets
├── test
└── tools
```
